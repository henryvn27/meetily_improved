# Archived audio pipeline experiment

This directory is retained only as design and migration reference for an unfinished second-generation audio pipeline. It is deliberately not registered from `src/lib.rs`, is not compiled into Meetily Improved, and must not be used for recording, transcription, packaging, or release verification.

The supported runtime is `src/audio/`. Any future attempt to revive this experiment requires a scoped Linear issue, an explicit compatibility plan for the existing Tauri commands and meeting data, platform audio tests, and a deliberate module registration change.
